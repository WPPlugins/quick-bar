== Description ==

A quicklook into Quick Bar - Popup Notification Bar

    ★ Create a light weight floating sidebar or popup notice bar
    ★ Full control on quick popup bar (sidebar) content
    ★ Standard WordPress content editor
    ★ Timeout based quick popup bar display
    ★ Display based on browsed number of pages
    ★ Css settings like color, z-index, border etc
    ★ Shortcode for displaying quick popup bar in specific pages
    ★ Display specificaly in pages/posts/home page

This plugin allows you to create a quick popup bar  (sidebar) with custom content in your site. You can place it like a floating sidebar on any side of the page using the available predefined positions.

You can customize the quick bar (Popup Notification Bar) display by configuring various settings such as display logic settings ( based on timeout after page load, based on number of pages browsed,  repeat interval) and style settings(z-index, color, border etc). The plugin supports automatic and manual (shortcode) display.

You can use the quickbar (Popup Notification Bar) to display any type of contents such as special promotions, subscription forms, social media icons, feedback forms and much more.

= 1. What is a "quick bar" or "sidebar" ? =

A quick bar  or sidebar is a floating dhtml popup bar used to display any content within a webpage, above the the page content. Sidebars are normally used to show social media links, subscription forms, contact forms, promotions, notifcations etc

= 2. The Quick Bar - Popup Notification Bar plugin is not working properly =

Please check the wordpress version you are using. Make sure it meets the minimum version recommended by us. Make sure all files of the `Quick Bar - Popup Notification Bar` plugin are uploaded to the folder `wp-content/plugins/`

= 3. What kind of contents can be placed in a Quick Bar - Popup Notification Bar ? =

You may  use the Quick Bar - Popup Notification Bar to display any type of contents such as social media icons, special promotions, subscription forms, contact forms, notifications  and much more.

= 4. Can I display Quick Bar - Popup Notification Bar popup in specific pages only ? =

Yes, you can do this by using the shortcode provided from settings page.

= 5. Does it work with caching plugins ? =

You can turn on compatibility with cache plugins from settings page. Please  note that since this method uses ajax, the popup will be a rendered a bit slower than in normal mode.

= 6. Can I create multiple Quick Bar - Popup Notification Bar popup boxes  for different pages ? =

To create multiple quickbars (sidebars), you need to purchase the premium version.
